📚 课堂签到系统

📸 照片录入要求（重要！）

✅ 正确示例

✓ 正面拍摄，面部完整清晰
✓ 光线充足均匀，无强烈阴影
✓ 表情自然，双眼睁开
✓ 不戴眼镜、帽子、口罩
✓ 背景简洁，无遮挡
✓ 照片清晰度高，文件大于100KB

❌ 错误示例

✗ 侧脸、低头、仰头
✗ 光线太暗或逆光
✗ 照片模糊不清
✗ 戴墨镜、口罩、帽子
✗ 多人合影或远距离拍摄
✗ 过度美颜或滤镜

💡 最佳实践：

  • 建议使用证件照或清晰的生活照
  • 人脸占照片的30-50%为最佳
  • 照片格式：JPG/PNG，分辨率建议800x1000以上
  • 批量导入时确保文件名格式：学号-姓名-班级.jpg
